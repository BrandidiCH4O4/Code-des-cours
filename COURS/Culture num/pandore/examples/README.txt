              PANDORE EXAMPLES


1. CONTENTS

   a. The batch files (*.bat)

These files are executable scripts.

   b. The .cpp files

These files are C++ files.

2. EXAMPLES

  a. The .bat files

To execute the batch files, just type the name.
For example:

	$ dog

  b. The .cpp files

     - To compile the .cpp files, copy the Makefile.msdos from
       directory doc\material and then type:

       visal C++:
        nmake -f Makefile.msdos exemple0.exe

      or dev-c++:
        make -f Makefile.msdos exemple0.exe
